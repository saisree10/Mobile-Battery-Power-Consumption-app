User Experience Program

The user experience enhancement program aims to create a better user experience established with the user more direct exchanges and communication, understanding the user the most realistic idea, and continuously push forward the user experience of the product. Participation in this program will not affect your use of our products and services. Participation in this program is not in any way interrupt or interfere with your use (such as pop-up dialog box, mail and telephone surveys). Whether or not to participate in this program, the performance of the products and services should be consistent.

Through user experience, we know:

1.user habits: The main including the user refreshments power saving, a function of frequency;

2.function usage: mainly includes user snack saving some functions, the button is clicked, the function is used, etc.. But does not include any privacy-related data generated by the user;

3.unexpected application error log: unexpected error procedures need to report the error log, so that we can accurately locate the cause of the error, to improve product quality;

Disclaimer: User Experience Program is only used to improve the product experience, we respect your privacy and will never share your information to any third party or used for other commercial purposes.
